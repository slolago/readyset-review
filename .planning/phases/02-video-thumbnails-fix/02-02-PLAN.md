<objective>
Route thumbnail uploads through the backend API to eliminate CORS failures in production.

Purpose: In production (Vercel), the browser PUTs the thumbnail blob directly to `storage.googleapis.com` via a signed URL. This fails because the GCS bucket CORS policy does not include the Vercel domain. By routing through `/api/upload/thumbnail`, the server-side GCS admin SDK uploads the blob with no CORS restrictions.
Output: New `/api/upload/thumbnail` route, updated client flow, `uploadBuffer` helper in gcs.ts
</objective>

**Step B: Create `src/app/api/upload/thumbnail/route.ts`**

Create a new API route that:
1. Authenticates the user using `getAuthenticatedUser(request)` from `@/lib/auth-helpers` (same pattern as signed-url/route.ts)
2. Accepts a POST with `multipart/form-data` containing:
   - `file` — the thumbnail Blob (image/jpeg)
   - `projectId` — string
   - `assetId` — string
3. Validates that projectId and assetId are present
4. Calls `canAccessProject(user.id, projectId)` to verify authorization
5. Verifies the asset exists and belongs to the user (fetch from Firestore `assets` collection, check `uploadedBy === user.id`)
6. Builds the GCS path: `buildThumbnailPath(projectId, assetId)`
7. Reads the file from FormData as an ArrayBuffer, converts to Buffer
8. Calls `uploadBuffer(gcsPath, buffer, 'image/jpeg')`
9. Builds the public URL: `getPublicUrl(gcsPath)`
10. Updates the asset document in Firestore: set `thumbnailUrl` to the public URL, set `thumbnailGcsPath` to gcsPath
11. Returns `{ thumbnailUrl, thumbnailGcsPath }` with status 200

Full implementation:

```typescript
import { NextRequest, NextResponse } from 'next/server';
import { getAuthenticatedUser, canAccessProject } from '@/lib/auth-helpers';
import { getAdminDb } from '@/lib/firebase-admin';
import { buildThumbnailPath, getPublicUrl, uploadBuffer } from '@/lib/gcs';

export async function POST(request: NextRequest) {
  const user = await getAuthenticatedUser(request);
  if (!user) return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });

  try {
    const formData = await request.formData();
    const file = formData.get('file') as File | null;
    const projectId = formData.get('projectId') as string | null;
    const assetId = formData.get('assetId') as string | null;

    if (!file || !projectId || !assetId) {
      return NextResponse.json({ error: 'file, projectId, assetId required' }, { status: 400 });
    }

    const hasAccess = await canAccessProject(user.id, projectId);
    if (!hasAccess) return NextResponse.json({ error: 'Forbidden' }, { status: 403 });

    const db = getAdminDb();
    const doc = await db.collection('assets').doc(assetId).get();
    if (!doc.exists) return NextResponse.json({ error: 'Asset not found' }, { status: 404 });

    const asset = doc.data() as any;
    if (asset.uploadedBy !== user.id) {
      return NextResponse.json({ error: 'Forbidden' }, { status: 403 });
    }

    const gcsPath = buildThumbnailPath(projectId, assetId);
    const arrayBuffer = await file.arrayBuffer();
    const buffer = Buffer.from(arrayBuffer);
    await uploadBuffer(gcsPath, buffer, 'image/jpeg');

    const thumbnailUrl = getPublicUrl(gcsPath);
    await db.collection('assets').doc(assetId).update({
      thumbnailUrl,
      thumbnailGcsPath: gcsPath,
    });

    return NextResponse.json({ thumbnailUrl, thumbnailGcsPath: gcsPath });
  } catch (error) {
    console.error('Thumbnail upload error:', error);
    return NextResponse.json({ error: 'Failed to upload thumbnail' }, { status: 500 });
  }
}
```
